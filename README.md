# Shop (React + TypeScript)
A shop application built with React, TypeScript, and modern web technologies.  

## Technologies  
- **Frontend**: React 18, TypeScript, Vite  
- **Styling**: CSS Modules
- **State Management**: Redux Toolkit / React Context  
- **API**: Mockapi
